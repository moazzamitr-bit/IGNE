# Design System

## Colors

Primary green: `#032c28`, `#063a34`, `#0a463e`, `#105047`.
Accent gold: `#c59b4a`, `#d6b66f`, `#e5cb91`.
Background ivory: `#fbf8f1`, `#f5f0e6`, `#ebe2d1`.
Burgundy is reserved for errors or rare emphasis.

## Typography

Persian uses Vazirmatn through `next/font`; English uses Inter. Persian body text uses generous line-height between 1.85 and 2.

## Layout

Main container width is capped at 1320px with responsive gutters from 20px to 48px.

## Components

Buttons use 6px radius, clear focus states, and restrained gold/green contrast. Cards use 8px radius or less, mild borders, and minimal shadow. Icons use a consistent line style through Lucide.

## Motion

Motion is limited to short reveal, hover lift, sticky header compaction, and carousel scrolling. Reduced motion is respected globally.
